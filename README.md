# Réplica de Facebook

Maqueta basica de FB donde se agrega lo explicado en cada clase.

## 🚀 Funcionalidades
- **Feed dinámico:** Podes escribir en el input y publicar posts reales en pantalla, si queres publicar sin tener nada escrito va a saltar alerta dando aviso.

- **Modo Claro / Oscuro Dinámico:** La interfaz permite alternar entre el modo oscuro (por defecto) y un modo claro inspirado en la paleta de colores real de Facebook. El navegador recuerda la preferencia de color del usuario (Sol/Luna) incluso después de cerrar la pestaña o recargar la página (`F5`) gracias al uso de LocalStorage.

- **Diseño responsive:** Interfaz adaptativa que funciona en navegadores de escritorio y dispositivos móviles mediante Media Queries.

## 🛠️ Tecnologías utilizadas
- [x] HTML5 (Estructura semántica)
- [x] CSS3 (Layouts con **Flexbox** y **Grid**)
- [x] JavaScript (Lógica de eventos y manipulación del DOM)
- [x] **Web Storage API (LocalStorage)**
